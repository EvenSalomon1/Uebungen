Git Cheatsheet

Basic

git add . -> fügt Änderung zur Staging-Area hinzu

git commit -m "first commit" -> erstellt eine neue commit mit einer Nachricht "first commit

git push -> lädt die Dateien der Staging-Area in das Github Repo

Branches

git branch -> zeigt alle vorhandenen Branches an

git branch BurgerMenu -> erstellt eine neue Branch, in dem Fall mit dem Namen BurgerMenu

git checkout BurgerMenu -> so wechselt man zur Branch „BurgerMenu

git checkout -b BurgerMenu -> legt eine neue Branch „BurgerMenu“ an und wechselt direkt dahin

git branch -d BurgerMenu -> Löscht die Branch BurgerMenu

Merge

  1. git checkout main -> wechselt zu main branch
  2. git merge BurgerMenu -> versucht die main branch mit BurgerMenu zusammenzuführen
  3. git add, git commit und git push
  4. git branch -d BurgerMenu -> Löscht die side branch „BurgerMenu“

Commits zurücksetzen

git log --oneline -> Zeigt alle commits mit den IDs an. Dann die ID auswählen bzw. kopieren zu der man zurückwill

git checkout B7398dj8 . -> setzt zurück auf den Stand von dem commit mit der ID B7398dj8.
